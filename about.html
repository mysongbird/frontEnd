About

SongBird is a project made as part of the Royal Hackaway Hackathon.

The aim for the project is to try and graph the popularity of songs over time in each country, based on the number of times they appear in Featured playlists on Spotify.

This is a front end web app which demonstrates some of the functionality of the backend API which has been created.
